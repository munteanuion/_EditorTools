Thank you for using the Missing Scripts Tool.

AisenCorporation
Version 1.0.0

Usage:
Editor's top navigation menu > Tools / Missing Script Tool.

-Find Missing Scripts in Entire Project: This button searches the entire project for GameObjects with missing scripts and logs the results.
-Open Current Scene: This button opens the currently active scene in the editor.
-Find Missing Scripts in Scene: This button scans the selected scene for GameObjects with missing scripts and logs the results.
-Save Scene: This button saves the currently active scene.
-Remove Missing Components: This button removes components with missing scripts from the selected GameObjects in the scene after user confirmation.
